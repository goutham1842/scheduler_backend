# Student Job Scheduler

This project is a FastAPI-based backend with a frontend interface for managing student jobs and scheduling tasks.

## Features
- Student login/signup
- Job creation
- Priority scheduling
- Deadline and urgency calculation
- FastAPI backend
- HTML/CSS frontend

## How to Run

Install dependencies:

pip install -r requirements.txt

Run backend:

uvicorn main:app --reload

Open browser:

http://127.0.0.1:8000/docs
